Multi-Service Application with Docker Networking

This project demonstrates deployment and operational support of a
multi-service application using Docker.

The focus is on container operations, service availability,
and troubleshooting rather than application feature development.

Key focus areas:
- Containerized application deployment
- Service-to-service communication
- Docker networking concepts
- Operational troubleshooting and support

This project aligns with DevOps Support, IT Operations,
and Application Support roles.
